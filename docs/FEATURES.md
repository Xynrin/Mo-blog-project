## 前台功能

### 📄 文章系统

**核心特性：**

- 📝 **Markdown 编辑** - 支持标题、列表、代码块、表格、引用等
- 🎨 **代码高亮** - 自动识别代码块语言，提供语法高亮
- 🏷️ **分类标签** - 文章支持分类和多标签
- 🖼️ **封面图** - 文章列表和详情页显示封面（16:9 推荐）
- 📊 **阅读统计** - 自动统计每篇文章的阅读次数
- 📱 **响应式** - 完美适配桌面、平板、手机等各种屏幕

**文章状态：**

- **草稿** - 仅管理员可见，不在前台展示
- **已发布** - 在首页、文章列表、分类页展示

### 🎬 多媒体支持

**图片嵌入：**

- 支持 JPEG、PNG、GIF、WebP、SVG、AVIF 格式
- 自动生成缩略图优化加载速度
- 响应式图片显示

**视频嵌入：**

- 🎥 **YouTube** - 直接嵌入 YouTube 视频
- 🎮 **Bilibili** - 支持 B 站视频
- 📹 **MP4 直链** - 自托管视频播放

### 🎨 作品集

**展示特性：**

- 卡片式布局，支持封面图、标题、描述
- 每个作品支持标签标识
- 项目链接跳转（GitHub、在线预览等）
- 拖拽排序控制显示顺序
- 可见性控制（展示/隐藏）

### 💬 联系表单

**表单功能：**

- 访客留言与反馈
- 防垃圾提交机制：
  - 时间戳验证
  - 蜜罐字段检测
  - 频率限制（同 IP 防刷）

**邮件通知：**

- ✉️ 新消息自动发送邮件提醒管理员
- 支持 QQ、163、Gmail 等邮箱
- 自定义 SMTP 配置

### 📡 RSS 订阅

自动生成两种订阅格式：

| 格式 | 地址 | 用途 |
|:---|:---|:---|
| RSS 2.0 | `/feed/rss` | 兼容性最好，适合大多数阅读器 |
| Atom 1.0 | `/feed/atom` | 现代标准，部分阅读器推荐 |

支持主流 RSS 阅读器：Inoreader、Feedly、NewsBlur 等

### 🏠 页脚功能

**页脚内容：**

- 自定义文字信息
- ICP 备案号展示（含超链接和盾牌图标）
- RSS 订阅链接
- 社交媒体链接（GitHub、Bilibili、YouTube 等）

## 后台功能

### 📊 仪表盘

**概览数据：**

- 文章总数、已发布、草稿统计
- 总阅读量
- 作品集和媒体文件数量
- 待处理消息数
- 最近 5 篇文章列表

### ✏️ 文章管理

**文章列表操作：**

- 新建文章
- 编辑已有文章
- 删除文章（二次确认）
- 发布/撤回文章
- 批量操作（计划中）

**编辑界面：**

- **标题** - 文章标题
- **Slug** - URL 友好路径，自动生成可手修改
- **分类** - 单分类选择
- **标签** - 多标签支持（输入后回车添加）
- **封面图** - 从媒体库选择或上传
- **摘要** - 文章简介（留空自动截取正文前 200 字）
- **Markdown 编辑器** - 支持以下功能：

| 功能 | 按钮 |
|:---|:---|
| 加粗 | **B** |
| 斜体 | *I* |
| 标题 | # |
| 列表 | ≡ |
| 代码块 | { } |
| 引用 | " |
| 光整理论 | ≡ |
| 图片 | 🖼 |
| 视频 | 🎬 |

### 🏷️ 分类管理

**分类属性：**

- **名称** - 分类显示名称
- **Slug** - URL 标识符
- **描述** - 分类说明文字
- **颜色** - 分类标识颜色
  - 同时控制文章标签按钮的背景色
  - 支持 16 进制和颜色名称
- **排序** - 数字越小越靠前

### 🎨 作品集管理

**作品属性：**

- **标题** - 作品名称
- **描述** - 作品介绍
- **封面图** - 作品展示图
- **项目链接** - 外部链接
- **标签** - JSON 格式标签数组
- **排序** - 显示顺序
- **可见性** - 展示或隐藏

### 📁 媒体库

**文件管理特性：**

- **上传方式** - 点击上传或拖拽文件
- **支持格式** - 图片、视频、PDF
- **大小限制**：
  - 图片：5 MB
  - 视频：50 MB
  - PDF：20 MB

**媒体库操作：**

| 操作 | 说明 |
|:---|:---|
| 网格/列表视图 | 切换显示方式 |
| 搜索 | 按文件名关键词搜索 |
| 类型筛选 | 图片 / 视频 / 文件 |
| 重命名 | 修改文件显示名称 |
| 删除 | 移除文件和缩略图 |
| 复制链接 | 获取文件访问 URL |

**图片处理：**

- 自动压缩优化
- 生成缩略图（-thumb 后缀）
- 按月归档（uploads/images/YYYY-MM/）

### 💌 消息管理

**消息属性：**

- 发送人姓名、邮箱
- 消息主题
- 消息内容
- 时间戳
- 读/未读状态

**消息操作：**

- 查看消息详情
- 标记已读/未读
- 删除消息
- 回复邮件（计划中）

## 系统功能

### ⚙️ 系统设置

#### 网站信息

| 设置项 | 说明 |
|:---|:---|
| 站点名称 | 显示在浏览器标签和页面头部 |
| 站点描述 | RSS 和 SEO 描述 |
| 作者名称 | 页脚和文章作者显示 |
| 作者简介 | 「关于我」页面内容 |
| 作者头像 | Cropper.js 实时裁剪，导出 512×512 JPEG |
| 每页文章数 | 文章列表分页数量 |
| 页脚文字 | 页脚左侧自定义文字 |
| Favicon | 浏览器标签图标（支持上传或 URL） |
| 备案信息 | ICP 备案号和官方链接 |

#### 社交媒体

可配置的社交平台：

- GitHub
- Bilibili
- YouTube
- X (Twitter)
- Gitee
- QQ
- 邮箱

#### 邮件服务

| 配置项 | 说明 |
|:---|:---|
| 邮件通知开关 | 启用/禁用邮件通知 |
| SMTP 服务器 | 邮件服务器地址 |
| SMTP 端口 | 587 或 465 |
| SMTP 用户名 | 邮箱地址 |
| SMTP 授权码 | 非邮箱登录密码 |
| 通知邮箱 | 接收通知的邮箱地址 |

### 🔐 账号安全

**账户管理：**

- **修改密码** - 输入当前密码和新密码（至少 6 位）
- **修改用户名** - 2-20 个字符，同步更新作者名称
- **JWT 认证** - 无状态 Token 认证
  - Token 默认有效期 7 天
  - 支持跨域认证

### 🎬 头像裁剪

**功能特性：**

- 使用 Cropper.js 库
- 实时预览
- 自由拖拽和缩放
- 导出 512×512 JPEG 格式
- 支持文件和 URL 上传

## 技术特性

### 🚀 性能优化

- **图片处理** - Sharp 自动压缩和缩略图
- **缓存策略** - 静态文件长期缓存
- **代码高亮** - 使用 highlight.js
- **数据库** - SQLite 零配置轻量级

### 🔒 安全特性

- **密码加密** - bcryptjs hash 加密
- **API 限流** - express-rate-limit 防刷
- **CSRF 保护** - 表单验证（计划中）
- **XSS 防护** - HTML 转义处理
- **SQL 防注入** - 参数化查询

### 📱 响应式设计

- 桌面端优化
- 平板端适配
- 手机端完整支持
- 流动布局字体缩放

### ✨ 动画效果

- 页面滚动揭示
- 卡片悬浮效果
- 粒子背景动画
- 平滑过渡动画

### 🌍 国际化准备

- 支持中英文切换（计划中）
- i18n 框架准备（计划中）
- 多语言模板（计划中）
