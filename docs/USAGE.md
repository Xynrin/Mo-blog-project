## 1. 首次使用

### 1.1 首次登录

1. 访问 `http://localhost:3000/admin`
2. 使用默认账号 `admin / admin123` 登录
3. **⚠️ 重要**：立即进入 **系统设置 → 账号安全** 修改密码

> 环境配置和安装步骤详见 [快速开始](QUICK_START.md) 和 [安装指南](INSTALLATION.md)

### 1.2 初始化网站信息

1. 进入 **系统设置 → 网站信息**
2. 修改站点名称、描述、作者信息
3. 配置社交媒体链接、Favicon、ICP 备案信息
4. 保存设置

## 2. 前台功能

### 2.1 首页

首页展示已发布的文章列表，包含：
- 文章封面、标题、摘要
- 分类、标签、发布时间
- 阅读量统计
- 分页浏览

### 2.2 文章列表

访问 `/articles` 查看所有已发布文章，支持：
- 按分类筛选
- 按标签筛选
- 按发布时间排序

### 2.3 文章详情

文章详情页展示：
- 完整的 Markdown 渲染内容
- 代码块语法高亮
- 嵌入的 YouTube / Bilibili 视频
- MP4 视频播放
- 分类与标签导航
- 阅读量统计（自动递增）

### 2.4 作品集

卡片式展示你的项目作品，包含：
- 封面图、标题、描述
- 项目链接（可跳转到 GitHub、在线预览等）
- 标签标识

### 2.5 联系表单

访客可通过 `/contact` 页面向你发送消息：
- 包含防垃圾保护（频率限制、蜜罐字段）
- 表单验证（邮箱格式检查）
- 可配置邮件通知

### 2.6 RSS 订阅

自动生成订阅源：
- **RSS 2.0**: `/feed/rss`
- **Atom 1.0**: `/feed/atom`

在 RSS 阅读器（Feedly、Inoreader 等）中添加上述链接即可订阅。

## 3. 后台管理

访问 `/admin` 进入后台，所有页面需要登录认证。

### 3.1 仪表盘

首页概览，显示：
- 文章总数、已发布/草稿数量
- 总阅读量
- 作品集和媒体文件数量
- 未读消息数
- 最近 5 篇文章

### 3.2 文章管理

**文章列表** (`/admin/articles`)：

| 操作 | 说明 |
|:---|:---|
| 写文章 | 创建新文章 |
| 编辑 | 修改现有文章 |
| 发布/撤回 | 切换发布状态 |
| 删除 | 删除文章（二次确认） |

**文章编辑页** (`/admin/article-edit`)：

| 字段 | 说明 |
|:---|:---|
| 标题 | 文章标题 |
| Slug | URL 路径（自动生成，可手修改） |
| 分类 | 选择一个分类 |
| 标签 | 输入标签名后回车添加 |
| 封面图 | 从媒体库选择或上传（推荐 16:9） |
| 摘要 | 文章简介（留空自动截取前 200 字） |
| 内容 | Markdown 编辑器 |

**编辑器工具栏：**

- **B** - 加粗文本
- **I** - 斜体文本
- **#** - 标题
- **≡** - 列表
- **{}** - 代码块
- **"** - 引用
- **🖼** - 插入图片（从媒体库选择）
- **🎬** - 插入视频（YouTube、Bilibili、MP4）

### 3.3 分类管理

**分类列表** (`/admin/categories`)：

| 字段 | 说明 |
|:---|:---|
| 名称 | 分类显示名称 |
| Slug | URL 标识符 |
| 描述 | 分类说明 |
| 颜色 | 分类标识颜色（同时控制标签背景） |
| 排序 | 数字越小越靠前 |

**操作：** 新增、编辑、删除分类

### 3.4 作品集管理

**作品列表** (`/admin/portfolios`)：

| 字段 | 说明 |
|:---|:---|
| 标题 | 作品名称 |
| 描述 | 作品介绍 |
| 封面图 | 展示图（推荐 16:9） |
| 项目链接 | 外部链接地址 |
| 标签 | JSON 数组格式：`["标签1", "标签2"]` |
| 排序 | 显示顺序 |
| 可见性 | 是否在前台展示 |

### 3.5 媒体库

**文件管理器风格** (`/admin/media`)：

**功能：**

- **上传** - 点击或拖拽上传文件
- **搜索** - 按文件名关键词搜索
- **筛选** - 按类型筛选（图片/视频/文件）
- **重命名** - 修改文件显示名称
- **复制链接** - 获取文件 URL
- **删除** - 移除文件和缩略图

**支持格式：**

| 类型 | 格式 | 限制 |
|:---|:---|:---|
| 图片 | JPG、PNG、GIF、WebP、SVG、AVIF | 5 MB |
| 视频 | MP4、WebM、OGG | 50 MB |
| 文件 | PDF | 20 MB |

**图片处理：**
- 自动压缩优化
- 自动生成缩略图（-thumb 后缀）
- 按月份归档：`uploads/images/YYYY-MM/`

### 3.6 消息管理

**消息列表** (`/admin/messages`)：

显示所有前台联系表单提交的消息：
- 发送人姓名、邮箱
- 消息主题、内容
- 提交时间
- 读/未读状态

**操作：**
- 标记已读/未读
- 删除消息
- 配置邮件通知（见系统设置）

## 4. 文章编写

### 4.1 Markdown 语法

编辑器支持标准 Markdown：

```markdown
# 一级标题
## 二级标题
### 三级标题

**加粗文本** *斜体文本* ~~删除线~~

- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2

> 引用文本

[链接文字](https://example.com)
![图片描述](图片地址)

`行内代码`

```javascript
// 代码块（指定语言）
const hello = 'world';
```

| 表头 1 | 表头 2 |
|:---:|:---:|
| 单元格 1 | 单元格 2 |
```

### 4.2 插入图片

**方式一：从媒体库插入**

1. 点击编辑器工具栏的「🖼 图片」按钮
2. 在弹窗中选择已有图片或上传新图片
3. 点击「插入」

**方式二：使用外链**

直接在 Markdown 中引用 URL：

```markdown
![图片描述](https://example.com/image.jpg)
```

### 4.3 插入视频

点击编辑器工具栏的「🎬 视频」按钮，支持：

| 平台 | 输入格式 |
|:---|:---|
| YouTube | `https://www.youtube.com/watch?v=VIDEO_ID` |
| Bilibili | `https://www.bilibili.com/video/BVxxxxxxxx` |
| MP4 直链 | `https://example.com/video.mp4` |

### 4.4 文章状态

- **草稿** - 仅管理员可见
- **已发布** - 在前台展示

## 5. 媒体库使用

### 5.1 上传文件

**上传方式：**
- 点击「上传」按钮选择文件
- 拖拽文件到上传区域
- 支持同时选择多个文件

**检查清单：**
1. 文件格式在允许列表中
2. 文件大小未超过限制
3. 磁盘空间充足

### 5.2 管理文件

**搜索文件：**
在搜索框输入文件名关键词

**按类型筛选：**
选择「所有」、「图片」、「视频」、「文件」

**重命名文件：**
1. 点击文件旁的重命名按钮
2. 输入新名称
3. 保存

**删除文件：**
点击删除按钮，文件及缩略图会被移除

**复制链接：**
获取文件的访问 URL，用于外部引用

### 5.3 图片处理说明

上传图片后，系统会：
- 自动压缩优化，减小文件体积
- 生成缩略图（`-thumb` 后缀）用于列表显示
- 按月份组织文件：`uploads/images/2026-04/`

## 6. 系统设置详解

访问 `/admin/settings` 配置博客。

### 6.1 网站信息

| 设置项 | 说明 |
|:---|:---|
| 站点名称 | 显示在浏览器标签和页面头部 |
| 站点描述 | 用于 SEO 和 RSS 描述 |
| 作者名称 | 显示在页脚和文章作者处 |
| 作者简介 | 「关于我」页面内容 |
| 作者头像 | 使用 Cropper.js 裁剪，导出 512×512 JPEG |
| 每页文章数 | 文章列表的分页数量 |
| 页脚文字 | 页脚左侧显示的自定义文字 |
| Favicon | 浏览器标签图标（支持上传或 URL） |
| 备案文本 | ICP 备案号（如「京ICP备xxxxxxxx号」） |
| 备案链接 | 备案号的链接（如工信部查询页面） |

### 6.2 社交媒体

配置社交平台链接，自动显示在页脚：

- GitHub
- Bilibili
- YouTube
- X (Twitter)
- Gitee
- QQ
- 邮箱

### 6.3 邮件服务

配置 SMTP 发送邮件通知：

| 配置项 | 说明 |
|:---|:---|
| SMTP 服务器 | 邮件服务器地址 |
| SMTP 端口 | 587（TLS）或 465（SSL） |
| SMTP 用户名 | 邮箱地址 |
| SMTP 授权码 | 非邮箱登录密码 |
| 通知邮箱 | 接收新消息的邮箱 |

**测试发送：**
点击「测试发送」按钮验证配置是否正确

> 详细的邮件配置（QQ、163、Gmail 等）见 [配置文档](CONFIGURATION.md#2-邮件服务配置)

## 7. 账号安全

### 7.1 修改密码

1. 进入 **系统设置 → 账号安全**
2. 输入当前密码和新密码（至少 6 位）
3. 点击「修改密码」

修改后需要重新登录。

### 7.2 修改用户名

1. 进入 **系统设置 → 账号安全**
2. 输入新用户名（2-20 个字符）
3. 点击「修改用户名」

修改后：
- 网站设置中的「作者名称」会同步更新
- 页脚和文章作者显示会更新
- 需要重新登录

### 7.3 JWT 认证说明

系统使用 JWT (JSON Web Token) 进行认证：

- 登录成功后返回 Token，存储在浏览器 localStorage
- Token 默认有效期 7 天
- Token 过期后需要重新登录
- 清除 localStorage 可强制退出登录
